Выберите платформу, чтобы установить VPN:

Следуйте инструкции:

1. Нажмите "Скачать приложение", установите его и нажмите "Готово".

2. Нажмите "Добавить подписку" и вернитесь на страницу подписки.

3. Нажмите "Добавить роутинг".

4. Нажмите "Настройки" -> "Подписка" -> активируйте "Обновить при открытии" -> вернитесь в главное меню.

5. Нажмите "Главная" -> "Роутинг" -> выберите "RUYT" -> активируйте "Включить" -> вернитесь в главное меню.

6. Включите VPN, нажав на синюю кнопку.
